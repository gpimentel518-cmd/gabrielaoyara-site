## Sobre o site

Site pessoal e portfólio de Gabriela Oyara — desenvolvedora de agentes de IA, estrategista digital, social media e artista. O site apresenta serviços, portfólio de casos reais, a agente de IA Ia-ra e o trabalho artístico (poemas e prosas).

## Como rodar localmente

Não é necessário servidor — basta abrir o arquivo no navegador:

```bash
# Clone o repositório
git clone (https://github.com/gpimentel518-cmd/gabrielaoyara-site)

# Abra no navegador
# Clique duas vezes em index.html
# ou arraste para a janela do navegador
```

Para um ambiente de desenvolvimento com live reload:

```bash
# Com Node.js instalado:
npx live-server

# ou com Python:
python -m http.server 3000
# Acesse: http://localhost:3000
```

## Destaques do projeto

**Ia-ra** (`iara.html`) — Landing page da agente de IA com hero animado, mockup de conversa WhatsApp em HTML/CSS puro, stats em destaque e CTA direto para WhatsApp.

**Portfólio** (`portfolio-gabriela-oyara.html`) — Cases reais com detalhamento de campanhas para FENAPRF, IPVA Alagoas, Rede Hiper Natacha e Unidor Saúde.

**Arte** — Seção dedicada a poemas e prosas, integrando a identidade artística ao posicionamento profissional.

## Vibe Coding

Este site foi desenvolvido com a abordagem Vibe Coding — iterando com IA em ciclos rápidos de prompt → geração → ajuste → publicação. Do conceito ao deploy em um único dia.
